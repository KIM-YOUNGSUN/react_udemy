# react,js
- 자바스크립트의 라이브러리 : 자바스크립트보다 코드가 간결하고 페이지 이동 시에 새로고침이 아닌 화면만 전환되기 때문에 리액트를 사용함
- 리액트는 컴포넌트 기반의 UI 라이브러리
- 리액트 대안 
  - Angular(완전한 컴포넌트 기반의 UI 프레임워크) 
    - 리액트보다 더 다양한 기능을 가지고 있음
  - Vuew.js(완전한 컴포넌트 기반의 UI 프레임워크) 
    - 앵귤러와 리액트를 섞어놓은 버전
    - 기능이 앵귤러보다는 적고 리액트보단 많음
    - 라우팅을 내장하고 있음
